# Teams app

Este directorio contiene un manifiesto inicial para empaquetar el bot en Microsoft Teams.

## Pendiente antes de usarlo en Teams real

- Sustituir `BOT-APP-ID-PLACEHOLDER` por el bot id real.
- Ajustar `validDomains`.
- Reemplazar `color.png` y `outline.png` por iconos definitivos.
- Comprimir `manifest.json`, `color.png` y `outline.png` en un zip para importarlo en Teams.
